# Presentation for local jibrel talk
## Prereqs:
You need [landslide](https://github.com/adamzap/landslide) for building one-file html presentation:
``` bash
python -m pip install landslide
```
## Usage:
``` bash
landslide -i -t theme/ -d index.html slides.md
```
